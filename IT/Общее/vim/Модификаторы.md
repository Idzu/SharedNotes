Если вдруг понадобится выполнить какую-то **команду несколько раз**, то для этого надо прописать перед командой надо **написать число**, сколько раз выполнить

##### Для обычных команд хватит просто ввести перед цифру
Например для удаления нескольких строк - **3dd**
Для перехода на 100 строку - **100j**

##### Для удаления 
Например для удаления двух слов - **d2w**
